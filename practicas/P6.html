5 Medidas de tendencia central

Las medidas de tendencia central resumen el “centro” de la distribución.

5.1 R base

mean(concentrado$ing_cor, na.rm=T)
median(concentrado$ing_cor, na.rm=T)
summary(concentrado$ing_cor)

5.2 Dplyr

concentrado |>
  summarise(
    Media = mean(ing_cor, na.rm=TRUE),
    Mediana = median(ing_cor, na.rm = TRUE))
NotaMedia vs. Mediana

Cuando la media es mayor que la mediana, la distribución tiene sesgo positivo: algunos valores muy altos “jalan” la media. En distribuciones de ingresos esto es casi siempre el caso.

6 Medidas de posición: cuantiles

Los cuantiles dividen la distribución en partes iguales. Por ejemplo los cuartiles (4 partes), los deciles (10 partes) y los percentiles (100 partes).

6.1 R base: summary() y quantile()

summary(concentrado$ing_cor)
# Cuartiles (Q1, Q2=mediana, Q3)
quantile(concentrado$ing_cor, probs = c(0.25, 0.50, 0.75), na.rm = TRUE)

# Deciles
quantile(concentrado$ing_cor, probs = seq(0.1, 0.9, by = 0.1), na.rm = TRUE)

6.2 dplyr: summarise() con quantile()

concentrado |>
  summarise(
    Q1  = quantile(ing_cor, 0.25, na.rm = TRUE),
    Q2  = quantile(ing_cor, 0.50, na.rm = TRUE),
    Q3  = quantile(ing_cor, 0.75, na.rm = TRUE),
    P10 = quantile(ing_cor, 0.10, na.rm = TRUE),
    P90 = quantile(ing_cor, 0.90, na.rm = TRUE)
  )

7 Medidas de dispersión

Las medidas de dispersión describen qué tan “esparcidos” están los datos alrededor del centro.

7.1 Varianza y desviación estándar

7.1.1 R base

var(concentrado$ing_cor,  na.rm = TRUE)
sd(concentrado$ing_cor,   na.rm = TRUE)

7.1.2 Dplyr

concentrado |>
  summarise(
    Varianza = var(ing_cor, na.rm = TRUE),
    Desv       = sd(ing_cor,  na.rm = TRUE)
  )

7.2 Rango y rango intercuartílico (RIC)

El rango corresponde al valor máximo menos el valor mínimo. Mientras que el rango intercuartílico corresponde al cuartil 3 menos el cuartil 1.

7.2.1 R base

max(concentrado$ing_cor,   na.rm = TRUE)-min(concentrado$ing_cor,   na.rm = TRUE)
IQR(concentrado$ing_cor,   na.rm = TRUE)

7.2.2 Dplyr

concentrado |>
  summarise(
    Rango = max(ing_cor, na.rm = TRUE)-min(ing_cor, na.rm = TRUE),
    RIC    = IQR(ing_cor, na.rm = TRUE)
  )

8 Medidas de sesgo (asimetría)

El coeficiente de asimetría (skewness) indica la dirección e intensidad de la cola de la distribución.

  • Sesgo = 0: distribución simétrica.
  • Sesgo > 0: sesgo positivo; la media > mediana.
  • Sesgo < 0: sesgo negativo; la media < mediana.
e1071::skewness(concentrado$ing_cor, na.rm = TRUE)
NotaInterpretación

Un sesgo positivo grande confirma que la distribución del ingreso tiene una cola larga hacia la derecha: la mayoría de los hogares se concentra en valores relativamente bajos, mientras que unos pocos tienen ingresos muy elevados.

9 Tabla descriptiva con summarise() y across()

9.1 Varias variables cuantitativas con across()

Cuando queremos aplicar las mismas funciones a múltiples columnas, across() evita repetir código.

# Seleccionamos un subconjunto de variables cuantitativas de ejemplo
concentrado |>
  summarise(
    across(
      .cols = c(ing_cor, gasto_mon, percep_ing),  
      .fns  = list(
        Media   = ~ mean(.x,   na.rm = TRUE),
        Mediana = ~ median(.x, na.rm = TRUE),
        DE      = ~ sd(.x,     na.rm = TRUE),
        Sesgo   = ~ e1071::skewness(.x, na.rm = TRUE)
      ),
      .names = "{.col}_{.fn}" #controla cómo se llaman las columnas resultantes del across().
    )
  ) |>
  tidyr::pivot_longer(everything(), # pasa de formato wide a long
                      names_to  = c("Variable", "Estadístico"),
                      names_sep = "_(?=[^_]+$)") |>   # parte el nombre de la columna en la última "_" para separar variable y estadístico
  tidyr::pivot_wider(names_from = "Estadístico", values_from = "value") |>  # regresa a wide, pero ahora cada estadístico es una columna y cada fila es una variable
  knitr::kable(digits = 2, caption = "Estadísticos descriptivos · variables seleccionadas")   # hace la tabla con 2 decimales y título

10 Detección de valores faltantes y extremos

10.1 Valores faltantes (NAs)

10.1.1 R base

sum(is.na(concentrado$ing_cor))
mean(is.na(concentrado$ing_cor)) * 100  # porcentaje

10.1.2 Dplyr

concentrado |>
  summarise(
    n_faltantes = sum(is.na(ing_cor)),
    pct_faltantes = mean(is.na(ing_cor)) * 100
  )

10.2 Valores extremos (outliers)

Una regla común: un valor es extremo si está más de 1.5 * RIC por encima del Q3 o por debajo del Q1 (criterio de Tukey).

Volvamos a calcular los cuartiles y el RIC.

q1  <- quantile(concentrado$ing_cor, 0.25, na.rm = TRUE)
q3  <- quantile(concentrado$ing_cor, 0.75, na.rm = TRUE)
ric <- q3 - q1

Ahora el límite inferior y superior.

limite_sup <- q3 + (1.5 * ric)
limite_inf <- q1 - (1.5 * ric)

¿Cuántos hogares están por encima del límite superior?

concentrado |>
  summarise(
    n_outliers_sup = sum(ing_cor > limite_sup, na.rm = TRUE),
    pct_outliers   = mean(ing_cor > limite_sup, na.rm = TRUE) * 100
  )

11 Filtrar valores

Podemos volver a hacer nuestra tabla pero con valores de ingreso mayor o igual a 0 y menor o igual al límite superior.

concentrado |>
  dplyr::filter(ing_cor >= 0, ing_cor <= 170320.5) |>
  dplyr::summarise(
    `N`              = dplyr::n(),
    `Media`          = mean(ing_cor,              na.rm = TRUE),
    `Mediana`        = median(ing_cor,             na.rm = TRUE),
    `Desv. estándar` = sd(ing_cor,                na.rm = TRUE),
    `Mínimo`         = min(ing_cor,               na.rm = TRUE),
    `Q1`             = quantile(ing_cor, 0.25,    na.rm = TRUE),
    `Q3`             = quantile(ing_cor, 0.75,    na.rm = TRUE),
    `Máximo`         = max(ing_cor,               na.rm = TRUE),
    `Asimetría`      = e1071::skewness(ing_cor,   na.rm = TRUE),
    `% faltantes`    = mean(is.na(ing_cor)) * 100
  ) |>
  gt::gt() |>
  gt::tab_header(
    title    = gt::md("**Estadísticos descriptivos del ingreso corriente trimestral**")
  ) |>
  gt::fmt_number(
    columns  = c(Media, Mediana, `Desv. estándar`, Mínimo, Q1, Q3, Máximo),
    decimals = 0,
    sep_mark = ","
  ) |>
  gt::fmt_number(
    columns  = c(Asimetría, `% faltantes`),
    decimals = 2
  ) |>
  gt::tab_source_note("Fuente: ENIGH 2024, INEGI. Ingreso en pesos corrientes trimestrales.") |>
  gt::tab_footnote(
    footnote  = "Criterio de Tukey: outlier si ing_cor > Q3 + 1.5*RIC.",
    locations = gt::cells_column_labels(columns = Asimetría)
  )

12 Resumen: R base vs. {dplyr}

Operación R base dplyr / tidy
Resumen general summary(df$var) skim(df)
Media mean(df$var, na.rm=TRUE) summarise(Media = mean(var, na.rm=TRUE))
Mediana median(df$var, na.rm=TRUE) summarise(Mediana = median(...))
Desviación estándar sd(df$var, na.rm=TRUE) summarise(DE = sd(...))
Cuantiles quantile(df$var, probs=...) summarise(Q1 = quantile(var, 0.25, ...))
Valores faltantes sum(is.na(df$var)) summarise(n_na = sum(is.na(var)))
