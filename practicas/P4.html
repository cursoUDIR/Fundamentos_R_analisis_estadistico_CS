1 Objetivos de la sesión

Al finalizar esta sesión podrán:

  • Explorar tablas o dataframes con un flujo reproducible usando dplyr y pipes (%>% o %>%).
  • Seleccionar, filtrar, ordenar, renombrar, agrupar y resumir variables con los verbos principales de dplyr.
  • Seleccionar columnas con helpers (por ejemplo ends_with(), contains(), matches(), where()) y con rangos del tipo var1:var2.
  • Aplicar transformaciones o renombres a múltiples columnas con mutate_at() y rename_at() (y reconocer su equivalente con across() / rename_with()).

5 2. Verbos principales de dplyr (mínimo para hoy)

5.1 2.1 select() + rangos var1:var2

El operador var1:var2 selecciona todas las columnas entre var1 y var2 en el orden en que aparecen en el dataframe.

# Ajusta los nombres a tu tabla
concentrado %>%
  select(folioviv:sexo_jefe)

5.2 2.2 select() + helpers: ends_with()

# Variables que terminan con "_pc" (ejemplo)
concentrado %>%
  select(ends_with("_cor"))

5.3 2.3 filter()

concentrado %>%
  filter(tam_loc == 1)

5.4 2.4 group_by() + summarise()

concentrado %>%
  group_by(tam_loc) %>%
  summarise(
    n = n(),
    media = mean(ing_cor, na.rm = TRUE),
    mediana = median(ing_cor, na.rm = TRUE)
  ) %>%
  arrange(desc(n))

6 3. Renombrar y transformar “en bloque” con rename_at() y mutate_at()

6.1 3.1 rename_at() (renombrar varias columnas)

Ejemplo: agregar un prefijo a un conjunto de variables seleccionadas por patrón.

concentrado2 <- concentrado %>%
  rename_at(
    vars(ends_with("_cor")),
    ~ paste0("pc_", .x)
  )

Equivalente moderno (referencia):

concentrado2 <- concentrado %>%
  rename_with(
    .fn = ~ paste0("pc_", .x),
    .cols = ends_with("cor")
  )

6.2 3.2 mutate_at() (transformar varias columnas)

Ejemplo: convertir a miles un conjunto de variables seleccionadas por rango o patrón.

concentrado3 <- concentrado %>%
  mutate_at(
    vars(ing_cor:otros_ing),   # 
    ~ .x / 1000
  )

Equivalente moderno (referencia):

concentrado3 <- concentrado %>%
  mutate(
    across(ing_cor:otros_ing, ~ .x / 1000)
  )
